# 프로젝트 소개

성취감을 통해 공부의 습관을 정착시킬 수 있는 개인 공부 관리 및 커뮤니티 서비스 사이트 제작


프로젝트 기간: 2024.08.13 ~ 2024.09.03

## 기술 스택

Frontend: JavaScript, React.js, scss

Backend: Express.js, PrismaORM

Database: PostgreSQL

공통 Tool: Git & Github, Discord
